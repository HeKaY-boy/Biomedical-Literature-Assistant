# Biomedical Literature Assistant

An AI-powered Retrieval-Augmented Generation (RAG) system that helps researchers retrieve and summarize biomedical literature from PubMed.

## Features

- Semantic Search
- Vector Database (FAISS)
- Embedding-based Retrieval
- LLM-powered Summarization
- Streamlit Interface

## Tech Stack

- Python
- Pandas
- Sentence Transformers
- FAISS
- Streamlit
